# Pure-Python Linux kernel Video4Linux2 (V4L2) bindings

## v4l2.uapi

v4l2.uapi namespace contains the kernel user-space API (uAPI).

## v4l2

v4l2 namespace contains wrappers to the uAPI to simplify the use of the uAPI.
